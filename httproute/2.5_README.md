#### Path Redirects

PATH Redirects는 HTTP Path Modifier를 사용하여 전체 경로 또는 경로 접두사를 바꿉니다. 예를 들어, 아래의 HTTPRoute는 경로가 /latte으로 시작하는 모든 www.f5bnk.com 요청을 /black로 302 리디렉션합니다. 특정 요구 사항에 따라 지원되는 상태 코드(301, 302, 303, 307, 308) 중 하나를 사용할 수 있습니다.

```yaml
apiVersion: gateway.networking.k8s.io/v1
kind: Gateway
metadata:
  name: path-redirect-gw
  namespace: web
spec:
  addresses:
  - type: "IPAddress"
    value: 192.168.48.202
  gatewayClassName: f5-gateway-class
  listeners:
  - name: http-path-redirect
    protocol: HTTP
    port: 80
    allowedRoutes:
      namespaces:
        from: "All"
      kinds:
      - kind: HTTPRoute
```

POST Method을 사용하는 특정 URL에 대해서 GET 요청으로 변환하는 HTTP Route를 생성합니다.

```yaml
apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: http-path-redirect
  namespace: web
spec:
  parentRefs:
  - name: path-redirect-gw
    sectionName: http-path-redirect
  rules:
  - matches:
    - path:
        type: PathPrefix
        value: /latte
    filters:
    - type: RequestRedirect
      requestRedirect:
        path:
          type: ReplacePrefixMatch
          replacePrefixMatch: /black
        statusCode: 302
  - backendRefs:
    - name: httpbin
      port: 8080
```

F5 BNK 2.1 버전에서는 HTTP Route의 302 Redirect 기능을 직접 지원하지 않기 때문에 이를 지원하기 위한 iRule을 생성합니다.

```yaml
apiVersion: "k8s.f5net.com/v1"
kind: F5BigCneIrule
metadata:
  name: irule-path-redirect
  namespace: web
spec:
  iRule: >
    when HTTP_REQUEST {
      if { [HTTP::uri] starts_with "/latte" } {
         set new_uri [string map {"/latte" "/black"} [HTTP::uri]]
         HTTP::respond 302 Location "https://[getfield [HTTP::host] ":" 1]$new_uri"
      } elseif { [HTTP::uri] starts_with "/green" } {
         HTTP::respond 302 Location "https://[getfield [HTTP::host] ":" 1]/green/tea"
        } else {}
    }
```

생성한 iRule을 Gateway에 적용합니다.

```yaml
apiVersion: "gateway.k8s.f5net.com/v1alpha1"
kind: BNKNetPolicy
metadata:
  name: netpolicy-path-redirect
  namespace: web
spec:
  extensionRefs:
  - kind: F5BigCneIrule
    name: irule-path-redirect
    group: k8s.f5net.com
  targetRefs:
  - kind: Gateway
    name: path-redirect-gw
    group: gateway.networking.k8s.io
    sectionName: http-path-redirect
```
