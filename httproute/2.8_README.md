#### Rewrites

Rewrite는 클라이언트 요청의 구성 요소를 수정한 후 프록시로 업스트림으로 전송합니다. URLRewrite 필터는 요청 호스트네임과/또는 경로를 변경할 수 있습니다. 예를 들어, HTTPRoute는 요청을 수락하여 설정한 Path와 Hostname 해더를 Rewrite하여 업스트림으로 전송합니다. 본 예제에서는 http://old.f5bnk.com/ip 요청을 new.f5bnk.com 및 /headers Path로 모두 Rewrite하는 것을 확인할 수 있습니다.

```yaml
apiVersion: gateway.networking.k8s.io/v1
kind: Gateway
metadata:
  name: rewrite-gw
  namespace: web
spec:
  addresses:
  - type: "IPAddress"
    value: 192.168.48.202
  gatewayClassName: f5-gateway-class
  listeners:
  - name: http-rewrite
    protocol: HTTP
    port: 80
    allowedRoutes:
      namespaces:
        from: "All"
      kinds:
      - kind: HTTPRoute
```

Path 기반의 Rewrite를 포함하는 HTTP Route를 생성합니다.

```yaml
apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: http-rewrite
  namespace: web
spec:
  parentRefs:
  - name: rewrite-gw
    sectionName: http-rewrite
  rules:
  - matches:
    - path:
        type: PathPrefix
        value: /headers
    filters:
    - type: RequestRedirect
      requestRedirect:
        path:
          type: ReplacePrefixMatch
          replacePrefixMatch: /ip
        statusCode: 302
  - backendRefs:
    - name: httpbin
      port: 8080
```

F5 BNK 2.1 버전에서는 HTTP Route의 Path 및 Header Modifiation이 지원되지 않기 때문에 아래와 같은 iRule을 적용하여 기능을 제공합니다.

```yaml
apiVersion: "k8s.f5net.com/v1"
kind: F5BigCneIrule
metadata:
  name: irule-rewrite
  namespace: web
spec:
  iRule: >
    when HTTP_REQUEST {
      if { [HTTP::uri] starts_with "/ip" && [HTTP::host] eq "old.f5bnk.com" } {
        set new_uri [string map -nocase {"/ip" "/headers"} [HTTP::uri]]
        HTTP::uri $new_uri
        HTTP::host "new.f5bnk.com"
      } elseif { [HTTP::uri] starts_with "/user-agent" && [HTTP::host] eq "old.f5bnk.com" } {
        set new_uri "/headers"
        HTTP::uri $new_uri
        HTTP::host "new.f5bnk.com"
      }
    }
```

생성한 iRule을 Gateway에 적용합니다.

```yaml
apiVersion: "gateway.k8s.f5net.com/v1alpha1"
kind: BNKNetPolicy
metadata:
  name: netpolicy-rewrite
  namespace: web
spec:
  extensionRefs:
  - kind: F5BigCneIrule
    name: irule-rewrite
    group: k8s.f5net.com
  targetRefs:
  - kind: Gateway
    name: rewrite-gw
    group: gateway.networking.k8s.io
    sectionName: http-rewrite
```
